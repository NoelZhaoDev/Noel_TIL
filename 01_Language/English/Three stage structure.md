# 🎯 Problem / Context 

> *因为中文的cognition structure是一个**Sequential mode**([[Sequential mode vs Synoptic mode]]). 在组织表达的时候，是很多孤立的点按照时间顺序连接起来的，无需有explicit的逻辑结构。这种认知结构是很难用英语express的。所以我总结了这个三段式结构。这是一个Synoptic mode，用更高的维度捕捉更整体的信息。*
> 

# ✅ Solution / Concept 

*   **Core Concept:** Three stage structure
*   **Explanation:**
    *   把一整个复杂的长事件拆成三部分来组织表达
	    * 我，为了解决什么问题，做了什么事情
	    * 我大概分了几步，每步做了什么事情
	    * 最终如何实现，核心方法是什么
