# 🎯 Problem / Context

> How should I express "我应该怎么写README文件合适" in English? What's the best way to express "覆盖范围" and "先后顺序" in English?

# ✅ Solution / Concept

*   **Core Concept:** Sequential mode vs Synoptic mode
*   **Explanation:**
    *   认知上对“如何做一件事情”提问，有两种方式。询问how to do 或者询问 what's the best way to do. 两种不同的提问代表了两类认知模式。Sequential mode关注的是动作的先后顺序，Synoptic mode关注的是整体或者结果。
    *   在developer风格的英语中，强调精准，简单、直观的表达风格里。鸟瞰模式要优于序列模式，这更不容易引来歧义和复杂的解释。
    * scope在语义上似乎是和覆盖范围，作用域等价的。
    * 按照规则，询问先做什么后做什么，就不如对结果sequence 询问，即What is the recommended sequence of sections?
