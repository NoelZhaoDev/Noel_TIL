# 单元音
## 1./ə/, /ɜː/ and /e/

>/ə/是不稳定的元音，论文里也没有采样数据。我也没有进行模仿。

/ɜː/是一个稳定的元音，位置在/ʌ/和/ɑː/的左上方，舌位更高更前。**我模仿的`目标共振峰`大概在F1=550，F2=1400**。这个元音和汉语拼音中/en/和/eng/韵母的前面部分类似，我自己的汉语里这个音的共振峰大概在F1=600，F2=1200。与英语十分接近，但二者仍然有显著的音质区别，不能替代。**/ɜː/音虽然稳定，却是英国英语中最奇怪最难发的音**。这个音对舌头精准度的控制远超所有其它的音，几乎在任何其它语言中都不存在对应或接近的音位。而美式英语的卷舌元音/ɚ/则对此简化，大大降低了发音的难度。

SSBE口音中/ɒ/和/e/音的F1大概也在500~600的区间内。要着重考略的是，为了与F1区间在600~800的/ʌ/和/ɑː/产生对立，/ɒ/, /ɜː/和/e/这组元音的F1尽量不要超过600。虽然相较于传统的rp发音，这几个音已经有高度下移的迹象了。但是不管怎么说，他们的F1也不应该与/ʌ/和/ɑː/产生重合。而把分辨音位的依据全部交给f2，尤其是在中后元音这个狭窄的空间内。

| Formant | /e/       | /ɜː/      | /ɒ/      |
| ------- | --------- | --------- | -------- |
| F1      | 500~600   | 500~600   | 500~600  |
| F2      | 1650~1800 | 1350~1550 | 800~1050 |

因此我选用了从/e/音向/ɜː/逼近的策略。/ɒ/音的具体发音因为还要和/ɔː/放在一组，所以挪到后面的章节再阐述。为什么选用/e/音而不是/ɒ/音来向/ɜː/音逼近呢？一开始我也确实这样做过。但是因为母语汉语本身不存在一个/ɒ/音位，以此为锚点会更加费力。其次则是舌头控制的方面，因为/ɜː/是一个最奇怪最难发的音，首先他的舌位要不低，是中高。这意味着你的舌根必须像向上抬起靠近软腭，同时嘴巴开口不能太大，否则F1会升高，轻易地超过600。同时它又是一个略微偏前的音，下颌可以稍微前移，舌头向前拉伸，甚至在舌前部向上颌隆起，类似前高元音/i/那样，但是又不能太靠前。如果不努力维持一个靠前的位置，F2会轻易地低于1300。整体而言它是略微偏向前高的方向的，比较符合前高元音的舌位。而不是后高元音。所以采用/e/音逼近更合理。

| 器官  | 前高(/i/)              | 前低(/æ/)                     | 后高(/u/)        | 后低(/ɑ/)        |
| --- | -------------------- | --------------------------- | -------------- | -------------- |
| 舌前部 | 抬起，不能收缩。舌面与上颚成阻      | 贴紧下颚，不能收缩。伴随舌根降低会略微抬起舌尖     | 向后收缩，与舌根一起贴紧软腭 | 向后紧缩，与舌根一起贴紧下颚 |
| 舌根  | 抬起，尽量靠近软腭，与舌前部组合构成拱形 | 沉底，舌根的高度会尽力降低，伴随舌前部的拉伸舌头会紧绷 | 抬起，紧贴软腭        | 紧缩，舌根高度尽量降低    |
| 下颌  | 下颌前移                 | 下颌前移                        | 不变             | 不变             |
| 嘴唇  | 开口度小                 | 开口度大                        | 开口度小，常伴有强烈圆唇   | 开口度较大          |

我的汉语/e/音即汉字欸的音，带有强烈的舌面向上隆起的特征，导致了我发这个音的F2剧烈上升，在2000以上。高度也因为开口度太大，且缺少舌根上升的特征，导致了F1经常超过600。知道了这些只要做出舌位上的调整，努力避免抬起舌面、同时注意开口度和舌根。就能发出目标范围内的共振峰。**我有一个小技巧是先发/i/音，保持舌根不变，让舌前部降下来发/e/音，这样能看到明显的舌头不在抬起，甚至是形成了凹槽，并发出一个低沉的相对/i/靠后的声音，这样就对了**。事实上这是一个对我而言发起来并不靠前的音，我的e音太靠前了，我甚至在感觉上要发一个靠后的，比/ɪ/还要靠后的音才是对的。

当把标准的/e/音发出后，做出/e/=>/ɛː/的长音，让F2慢慢降低，找到一个稍微靠前一点点的的舒适位置，标准的/ɛː/音也就能固定下来了。让F2慢慢降低，也就是在保持开口度与舌根的高度的情况下，让舌头慢慢向后收缩。**降低F2的时候要确保舌根不要动，否则一旦F1升高了，就毫无意义了**。

用i和k逼近，在舌根紧紧上抬的情况下发出，不用那么用力挤声音。它需要前元音的舌头前度甚至略微拱起的舌面，以及高元音那样抬起的舌根。舌根位置特别特别高。但不应太紧绷，这个声音没必要那么紧张。注意不能把它发成舌根下降的音，也不是隆起舌根后下降，而是舌根轻微朝向软腭隆起的音，同时注意舌头前部。

**注意，不要试图用schwa的高度来发，要用汉语里的那个后元音/ɤ/这两者高度是一致的，shcwa的高度太低了**。

汉语母语中不存在的音来作为锚点去逼近呢？原因有两点，第一是如果使用母语/en/或/eng/中的e音来靠近的话，因为两者的共振峰十分接近，且这里汉语的e音从不单独出现，只在鼻韵母/en/或/eng/中出现，我是很难把这个音发稳定，并且精准控制这个音向上且向前滑动的。每次调整要么就是调整过度，要么就是根本没动，实现起来相当困难。第二点就是从生理角度考虑，如果从后元音开始，稳定住F1不变，也就是舌根的位置不变。那么单独控制舌前部来改变F2，是更加容易实现的。通过/ɒ/音作为基准，维持舌根部不动，发一个较前的/ɜː/音，很快就能做到。

或许从schwa开始前移下颌略微张嘴，发一个较低的从喉咙出来的音也是不错的选择。

通过前移下颌可以有效升高F2，而不仅仅是依靠舌头前部的移动。从原理上来看，针对F2在口腔内产生的两个极值点，舌中后部位以及舌前部。舌前部位于极大压强处，下颌前移意味着口腔前部成阻，实际上是前腔缩短。从多级管道来看缩短的前腔也可以有效升高f2.

相较于论文中提供的数据，我观测到的/ɜː/音的位置有轻微的后移和下降。但仍未偏移太多，可以认为在SSBE中，这个音没有产生变化。

## 2./ʌ/, /ɑː/ and /æ/

传统上认为/ʌ/和/æ/之间或者/ʌ/和/ə/之间容易混淆，并把它们视为一组。但是经过我的测量，在现代SSBE口音中，/ʌ/和/ɑː/是音质最接近的一组元音，二者的F1几乎没有差别，F2的相差也只有200甚至更低。把同为低元音的/æ/和它们放在一起，是更好的选择。

/ʌ/和/ɑː/的F1都分布在600到800。有人把/ʌ/发得略高，有人把/ɑː/发得略高，也有人把两者发得几乎一样高。因此在模仿时，我也不进行二者高度上的区分。但是值得一提的是，/æ/的F1相较于保守发音有大幅度下降，800以上、900甚至1000以上的F1我都见过。为了能够把/æ/和/ʌ/, /ɑː/区分开，我选择把/æ/发到900以上，但避免超过1000，因为这样会带有更多的南部口音。

从数据分布来看，我认为/ʌ/的F2要不能低于1200，/ɑː/的F2要不能高于1100。这是能够辨别二者的最低要求，事实上二者的音位区别还体现在音长和稳定度上。单靠这100赫兹的前后区别是很难分清这两个音位的。/æ/的F2应该在1500以上，如果能够发得更靠前也是可以的。但我没有测量到前的数据，我也发不出更靠前的声音。维持在1500以上就足够了。也就是说这三者的共振峰区别如下。

| Formant | /æ/    | /ʌ/     | /ɑː/    |
| ------- | ------ | ------- | ------- |
| F1      | 900左右  | 600~800 | 600~800 |
| F2      | 1500左右 | 不低于1200 | 不高于1100 |

可以看出三者基本成水平关系从前向后。选取的锚定单词为back，cut，car

## 3./iː/ and /ɪ/

先考虑/iː/的发音。前高元音的位置并不难找，龇牙并且微笑就能发出。/ɪ/的位置则要更低更后。处理方法是把舌前部不要抬起尽力后缩来降低F2。并尽力维持舌根不变。不然F1就升得太厉害了。

因为英音韵律的问题，可能会出现把/iː/发成/ɪi/的滑动的情况。尤其是在词尾的开音节。但是这并非刻意发出的滑动。不能发得太刻意。

| Formant | /iː/   | /ɪ/    |
| ------- | ------ | ------ |
| F1      | 300左右  | 400左右  |
| F2      | 2200左右 | 1800左右 |

## 4./uː/ and /ʊ/

SSBE口音内这一组元音的主要问题在于严重前移，实际观测到的数据比论文中的数据还要更加严重。甚至达到了/y/的夸张程度。

| Formant | /uː/      | /ʊ/    |
| ------- | --------- | ------ |
| F1      | 300左右     | 400左右  |
| F2      | 1700到1800 | 1400左右 |
## 5./ɔː/ and /ɒ/

这一组后元音有一个稍微有意思的点。虽然和英音无关，但是还是稍微提几句。美式发音里没有/ɒ/，这一音位在美式英语里被合并到了/ɔː/，但在西海岸口音的影响下，/ɔː/又和/ɑː/发生了合并。又因为实际上的美音/ɔː/音质更接近于 英音的/ɒ/。导致中式英语里对这一组元音的发音产生了极大的偏移。实际上二者都被合并到了美音的/ɔː/。

| Formant | /ɔː/  | /ɒ/   |
| ------- | ----- | ----- |
| F1      | 400左右 | 550左右 |
| F2      | 750左右 | 950左右 |

# 双元音
## 1./əʊ/

/əʊ/起始位置大概是在/ə/偏后一点，终点位置大概是在 /uː/偏后一点
## 2./eɪ/ and /ɔɪ/

/eɪ/起始位置大概是在/e/偏上，终点位置大概是在 /iː/偏后偏低一点
/ɔɪ/暂时没有数据支持
## 3./aɪ/ and /aʊ/

/aɪ/起始位置大概是在/ʌ/, /ɑː/之间，终点位置大概是在 /ɪ/偏前一点
/aɪ/起始位置大概是在/æ/偏后偏高一点，终点位置大概是在 /ʊ/和/ɔː/之间，完全偏向一侧的发音也存在。
## 4./ɪə/, /ʊə/ and /eə/

三者存在很明显的音变
/ɪə/->/iə/ or /i/
/ʊə/->/ɔː/
/eə/->/ɛː/
